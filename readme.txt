Agrega las Provincias de Ecuador a WooCommerce

== Description ==

Dentro de la instalación de Woocommerce cambia el campo State/Estado y lo cambia por Provincias, así como agrega la lista de las 24 Provincias de Ecuador

== Installation ==

Para instalar este plugin haga lo siguiente:

1. Suba `woocoomerce-provincias-ecuador.php` a la carpeta `/wp-content/plugins/`
1. Active el plugin a traves del menú 'Plugins' en WordPress
